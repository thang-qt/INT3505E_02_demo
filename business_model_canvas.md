# **Business Model Canvas – Ecommerce API**

## **1. Customer Segments**
- Developers tích hợp API  
- Merchants/Sellers  
- Dịch vụ 3rd-party (shipping, payment, ERP)  
- Mobile app developers  

## **2. Value Propositions**
- Truy cập dữ liệu sản phẩm/đơn hàng chuẩn hóa  
- Đồng bộ tồn kho real-time  
- Mở rộng hệ sinh thái đối tác  
- Developer experience tốt (docs, sandbox, API keys)  
- Hỗ trợ tăng doanh thu cho seller  

## **3. Channels**
- Developer Portal  
- Seller Dashboard (quản lý API keys)  
- Email/Webhook onboarding  
- Cộng đồng dev (Discord, GitHub)  

## **4. Customer Relationships**
- Self-service  
- Community support  
- Hỗ trợ theo tier (free → enterprise)  
- Quickstart guides, SDK  

## **5. Revenue Streams**
- Freemium  
- Pay-per-call / tier-based  
- Gói API premium cho merchant  
- Enterprise SLA  
- Partnership revenue từ dịch vụ tích hợp  

## **6. Key Resources**
- API backend, API gateway  
- Database  
- Developer portal & docs  
- Đội dev + DevOps  
- Monitoring & analytics  

## **7. Key Activities**
- Thiết kế & phát triển API  
- Duy trì docs & SDK  
- Bảo mật API key, auth, rate-limit  
- Monitoring (latency, error rate, traffic)  
- Onboard đối tác (shipping/payment)  

## **8. Key Partners**
- Payment gateways  
- Logistics partners  
- ERP/CRM/POS  
- Cloud providers  

## **9. Cost Structure**
- Server, DB, CDN, gateway  
- Monitoring & logging  
- Phát triển & bảo trì  
- Bảo mật  
- Marketing/onboarding seller
